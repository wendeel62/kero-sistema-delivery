# Componentes Organisms

Este diretório contém componentes compostos do sistema Kero.

## Estrutura

Cada pasta representa um domínio de功能:
- Dashboard/ - Componentes do dashboard
- Pedidos/ - Componentes de pedidos
- Financeiro/ - Componentes financeiros

## Padrão

Cada componente deve ter:
- Arquivo principal: NomeDoComponente.tsx
- Teste: NomeDoComponente.test.tsx
- Types: types.ts (se necessário)
