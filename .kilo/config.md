# Kero Project Configuration

This file configures the Kilo CLI for this project.

## Project Settings
- name: kero-project
- type: webapp (React/TypeScript with Supabase)
- version: 1.0.0

## Agents
The following agents are available in this project:
- architect: Software architecture and design
- dev: Full-stack development
- qa: Testing and quality assurance
- pm: Project management
- po: Product ownership
- sm: Scrum mastery
- analyst: Business and technical analysis
- devops: Infrastructure and deployment
- ux-design-expert: User experience design
- squad-creator: Multi-agent squad design

## Skills
The following skills are available:
- frontend-design: Production-grade frontend interfaces
- web-design-guidelines: Web UI best practices
- angular-component: Angular component patterns
- angular-di: Angular dependency injection
- create-pull-request: GitHub PR creation
- file-organizer: File organization
- skill-creator: Skill development guide

## Commands
- npm run dev: Start development server
- npm run build: Build for production
- npm run lint: Run ESLint
- npm run typecheck: Run TypeScript check

## Activation
To activate an agent, use:
- /agents/agent-name (e.g., /agents/architect)
- @agent-name (e.g., @dev)

## Workflow
1. Use analyst to gather requirements
2. Use architect to design the solution
3. Use dev to implement
4. Use qa to test
5. Use devops to deploy
