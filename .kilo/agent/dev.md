# dev - Desenvolvedor

Agent para implementação de código, debugging e desenvolvimento.

## Fonte
Este agent herda de `.aiox-core/development/agents/dev.md`

## Comandos Disponíveis
- `*dev` - Ativar modo desenvolvimento
- `*build` - Executar build
- `*lint` - Executar lint
- `*test` - Executar testes

## Regras do Projeto
- Siga a Constitution em `.aiox-core/constitution.md`
- Sempre use tenant_id em todas as queries Supabase
- Execute `npm run build` antes de fazer commit
