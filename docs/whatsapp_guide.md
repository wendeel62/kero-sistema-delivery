# Guia do Novo WhatsApp Kero

Bem-vindo à nova interface do WhatsApp integrada ao Kero! Agora você pode gerenciar suas conversas e pedidos de forma centralizada e com auxílio de Inteligência Artificial.

## 🚀 Como Começar

1. **Conexão**: Acesse a página do WhatsApp no menu lateral.
2. **QR Code**: Caso a instância não esteja conectada, um QR Code será exibido. Escaneie-o com seu celular (WhatsApp -> Aparelhos Conectados).
3. **Sincronização**: Suas conversas aparecerão na barra lateral esquerda.

## 🤖 Automação com IA (Groq)

O sistema agora conta com um atendente virtual inteligente:
- **Extração de Pedidos**: Quando um cliente envia um pedido em linguagem natural (ex: "Quero 2 pizzas"), a IA identifica os itens, quantidades e cria o pedido automaticamente no sistema.
- **Confirmação**: O cliente recebe uma confirmação imediata.
- **Intervenção Humana**: Você pode assumir a conversa a qualquer momento enviando uma mensagem manualmente pela interface.

## 🎨 Interface Premium

- **Chats em Tempo Real**: Veja quem está online e as mensagens não lidas.
- **Micro-animações**: Transições suaves e efeitos de vidro (glassmorphism) para uma experiência premium.
- **Atalhos**: Use `Enter` para enviar e `Shift + Enter` para pular linha.

---
*Dica: Mantenha seu celular conectado à internet para garantir a estabilidade da Evolution API.*
